### Google Maps ŌĆö auth model (no user OAuth today)

The **Maps SDK for iOS** authenticates with a **server-side API key** bundled in the app (see **Secrets / setup** above). It does **not** sign the user into a Google account or send dive log data to Google for basic tile + marker display.

**Not required for current spike:** Google **OAuth consent screen** or ŌĆ£share data with Google MapsŌĆØ user flow.

**Revisit before adding:** Google **Places** (autocomplete / search), **Routes**, **Geocoding** as a billed Google REST API, **Google Sign-In**, or any API that reads a userŌĆÖs Google account or sends user-generated dive content to Google backends ŌĆö those may need a privacy-policy update, OAuth consent configuration, and/or additional API enablement. Track in **`cursor/todo.md`** ŌåÆ **Google Maps / Google Cloud**.

### Fishial.AI ŌĆö optional cloud fish identification

GoDive depends on **[Fishial.AI](https://fishial.ai/)**ŌĆÖs hosted **Recognition API v2** for the manual **Identify fish** flow on dive media (pinkŌåÆpurple gradient **sparkles** beside **+** on the **Marine life** tag sheet leading toolbar). The feature is **off unless configured**: **`FishialSecretsBootstrap`** loads **`ClientID`** / **`ClientSecret`** from gitignored **`GoDiveMVP/Config/FishialSecrets.plist`** (free developer keys from [portal.fishial.ai](https://portal.fishial.ai)).

**Selection UX:** photos open the crop editor on a fast local preview (**`.fastFormat`**, **1024** edge); videos scrub on the lighter **`.homeCarousel`** stream with a live-updating **`AVAssetImageGenerator`** still preview (not keyframe-snapped **`AVPlayer`** seeks) that refreshes as the slider drags ŌĆö requests are coalesced (**`FishialVideoScrubFrameRequestCoalescer`**: one decode at a time, newest fraction runs next) at a small **┬▒0.05 s** tolerance so intermediate frames show instead of only the frame after the finger lifts ŌĆö and duration from **`PHAsset`** metadata ŌĆö full iCloud originals load only when the user taps **Identify** / **Continue**. **What leaves the device:** one **JPEG still** per identify (user-cropped full-quality photo export or cropped scrubbed video frame) via **`POST /v2/recognize`**, plus optional dive coordinates in the **`Fishial-Location-Lat-Lon`** header when the associated **`DiveActivity`** has a resolvable map coordinate (linked site, entry GPS, or catalog name match). **No automatic batching** ŌĆö the user must tap **Identify** after framing the fish in the crop editor (videos: scrub ŌåÆ **Continue** ŌåÆ crop ŌåÆ **Identify**).

**What stays local:** PhotoKit originals, the full dive log, and Field Guide catalog data. Fishial returns species candidates only; GoDive **fuzzy-matches** those scientific names against the bundled **`MarineLife`** catalog (**`FishialMarineLifeCatalogMatching`**) and shows catalog thumbnails in the review sheet. When the user confirms a match (single result) or picks one from multiple, GoDive creates a **`SightingInstance`** tag on that catalog species (**`MarineLifeSightingRecorder.tagSpecies`**) and saves the catalog scientific name on **`DiveMediaPhoto.fishialConfirmedSpeciesName`**; the matching species oval at **medium** / **large** shows an in-chip **sparkles** badge.

**Operational notes:** requires network and a signed-in profile to save catalog tags; subject to Fishial account credits, API availability, and their [API terms / privacy policy](https://docs.fishial.ai/api). Client code: **`FishialAPIClient`**, **`FishialObservationLocation`**, **`DiveMediaFishialIdentification`**, **`FishialMarineLifeCatalogMatching`**, **`DiveMediaFishialIdentificationStorage`**, **`FishialIdentificationReviewPresentation`**. Revisit privacy copy and App Store disclosures before shipping identify broadly or enabling background/auto-identify.

**Manual media tagging:** **`DiveMarineLifeTagPickerSheet`** (blue **`.large`** Cancel / **+** / Done modal) stages catalog picks until trailing **Done** commits **`MarineLifeSightingRecorder.tagPendingSpecies`**; **Cancel** discards staged tags; **+** opens **`FieldGuideMarineLifeAddSheet`** and stages the new species; buddy tag picker uses the same blue overview-panel chrome.
